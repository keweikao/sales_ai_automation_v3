# /todo-summary - 待辦統計總覽

## 描述
快速查看團隊或個人的待辦統計，包括逾期、今日、本週的待辦數量和完成率。適合主管快速了解團隊執行狀況。

## 使用方式
```
/todo-summary                    # 團隊總覽（預設本週）
/todo-summary --today            # 今日統計
/todo-summary --week             # 本週統計
/todo-summary --month            # 本月統計
/todo-summary --user 王大明      # 特定業務統計
/todo-summary --overdue          # 逾期待辦詳情
/todo-summary --ranking          # 團隊排行榜
```

## 執行流程

### 步驟 1: 解析參數

| 參數 | 預設值 | 說明 |
|------|--------|------|
| 時間範圍 | --week | 統計的時間範圍 |
| --user | 全部 | 指定特定業務 |
| --overdue | false | 顯示逾期詳情 |
| --ranking | false | 顯示團隊排行 |

### 步驟 2: 查詢資料庫統計

**2.1 總體統計：**
```sql
SELECT
  COUNT(*) as total,
  COUNT(CASE WHEN status = 'pending' THEN 1 END) as pending,
  COUNT(CASE WHEN status = 'completed' THEN 1 END) as completed,
  COUNT(CASE WHEN status = 'won' THEN 1 END) as won,
  COUNT(CASE WHEN status = 'lost' THEN 1 END) as lost,
  COUNT(CASE WHEN status = 'pending' AND due_date < CURRENT_DATE THEN 1 END) as overdue
FROM sales_todos
WHERE due_date BETWEEN ? AND ?;
```

**2.2 按用戶統計：**
```sql
SELECT
  u.id as user_id,
  u.name as user_name,
  COUNT(*) as total,
  COUNT(CASE WHEN t.status = 'pending' THEN 1 END) as pending,
  COUNT(CASE WHEN t.status = 'completed' THEN 1 END) as completed,
  COUNT(CASE WHEN t.status = 'pending' AND t.due_date < CURRENT_DATE THEN 1 END) as overdue
FROM sales_todos t
JOIN users u ON t.user_id = u.id
WHERE t.due_date BETWEEN ? AND ?
GROUP BY u.id, u.name
ORDER BY completed DESC;
```

**2.3 逾期詳情：**
```sql
SELECT
  t.id,
  t.title,
  t.due_date,
  t.postpone_history,
  o.company_name,
  o.customer_number,
  u.name as user_name
FROM sales_todos t
LEFT JOIN opportunities o ON t.opportunity_id = o.id
JOIN users u ON t.user_id = u.id
WHERE t.status = 'pending' AND t.due_date < CURRENT_DATE
ORDER BY t.due_date ASC;
```

### 步驟 3: 計算指標

- **完成率** = completed / (completed + pending) * 100
- **逾期率** = overdue / pending * 100
- **平均改期次數** = SUM(postpone_count) / total

## 注意事項

1. **權限** - 一般業務只能查看自己的統計，主管可查看團隊
2. **時區** - 所有時間為 UTC+8（台灣時間）
3. **即時性** - 資料直接從資料庫查詢，無快取延遲
